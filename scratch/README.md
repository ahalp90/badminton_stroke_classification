# scratch/

Shared work-in-progress catchall. Drop quick notes, sweep configs, plot
output, and one-off scripts here while they're still fluid.

Matured work moves out: live docs to `docs/`, live scripts to `scripts/`, run
artefacts to `experiments/`. Periodic sweeps clear the rest. For private
scribbles that should never enter git, use `local_scratch/` at repo root
(gitignored).
